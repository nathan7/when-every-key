# when-every-key

  promise aggegation for objects

## Installation

    $ component install nathan7/when-every-key

  or

    $ npm install when-every-key

## API

### whenEveryKey :: { (key): Promise a } -> Promise { (key): a }

  Takes an object, resolves all its values.
